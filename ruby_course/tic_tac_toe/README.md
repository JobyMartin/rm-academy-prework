# Tic-Tac-Toe

A terminal-run Tic-Tac-Toe game.

## Getting Started

1. Clone the repository
2. Change directory to the project folder:
```bash
cd ruby_course/tic_tac_toe
ruby main.rb
```
3. Play the game:
```bash
ruby main.rb
```

## Usage

* Enter quadrant numbers to place pieces (1-9 grid layout)
* Place 3 in a row to win
