# Mastermind

A terminal-run Mastermind game.

## Getting Started

1. Clone the repository
2. Change directory to the project folder:
```bash
cd ruby_course/mastermind
```
3. Play the game:
```bash
ruby main.rb
```

## Usage

* Enter colors to guess secret code
* Guess the code within 12 tries to win

## Features

* Game timer
* User option to be the code maker or the code breaker

## Technologies Used

* Ruby
