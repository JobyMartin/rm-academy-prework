# Hangman

A terminal-run Hangman game.

## Getting Started

1. Clone the repository
2. Change directory to the project folder:
```bash
cd ruby_course/hangman
```
3. Play the game:
```bash
ruby main.rb
```

## Usage

* Enter letters to guess the word
* Guess the word before getting 8 letters wrong to win

## Features

* Abiltiy to serialize (save) games and reload them

## Technologies Used

* Ruby
* YAML
