# RoleModel Academy Prework

This repository contains my completed prework for the RoleModel Craftsmanship Academy.

## Structure

- Each **course** is organized into its own folder in the root directory  
- Each **project** is contained within its course folder  
- Individual projects include their own README with details and instructions  

## Running Locally

Clone the repository:
```bash
git clone https://github.com/JobyMartin/rm-academy-prework.git
cd rm-academy-prework
```
Refer to each project's README for specific setup and run instructions.

## Notes
- HTML/CSS/JavaScript projects can be run in the browser using a local server
- Some projects are committed in bulk due to past completion
