# Etch-a-Sketch

A browser-based drawing app.

## Features

* Draw by hovering over the grid
* Adjustable grid size
* Clear/reset button

## Getting Started

1. Clone the repository
2. Open the project folder
3. Run on a local server or open `index.html`

## Usage

* Move your mouse over the grid to draw
* Use controls to resize or reset the grid

## Technologies Used

* HTML
* CSS
* JavaScript
