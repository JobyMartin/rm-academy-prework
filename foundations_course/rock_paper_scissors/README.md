# Rock Paper Scissors

A simple Rock Paper Scissors game played against the computer in the browser.

## Features

* Player vs computer gameplay
* Randomized computer choices
* Score tracking

## Getting Started

1. Clone the repository
2. Open the project folder
3. Run a local server or open `index.html`

## Usage

* Click Rock, Paper, or Scissors
* The computer will make a random choice
* The winner is displayed on the screen

## Technologies Used

* HTML
* CSS
* JavaScript
